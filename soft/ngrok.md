## ngrok

> References:
>
> https://ngrok.com/docs



TLDR:

1. Sign up, get token and install it

```bash
ngrok authtoken <YOUR_AUTHTOKEN>
```

1. Expose a web server on port 80 of your local machine to the internet

```bash
ngrok http 80
```

1. Open http://localhost:4040
