### ASDF

> Referenes:
>
> https://asdf-vm.com/#/

It is like `gvm`, `nvm`, `rbenv` & `pyenv` (and more) all in one!



Installation:

```
https://github.com/ohmyzsh/ohmyzsh/tree/master/plugins/asdf
```

Plugins:

```
asdf plugin add <name>
asdf plugin list
```

Versions:

```
asdf list all <name>
asdf install <name> <version>
asdf global <name> <version> [<version>...]
asdf current
```
